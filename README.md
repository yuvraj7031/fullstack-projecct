# AI Blog App Backend

Backend service for an AI-powered blogging platform built with MERN stack.

## Features

- User authentication with JWT
- Blog creation/edit/delete
- Like and comment system
- Image upload with Cloudinary
- Gemini AI integration for blog assistance
- REST APIs tested with Postman

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Cloudinary
- Gemini API

## Run locally

npm install

Create .env file

npm run dev
